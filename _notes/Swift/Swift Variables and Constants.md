---
title: Swift Variable and Constants
---

### Main Idea

- Store data
    - Variable 
        - Data to change over time
            - Var name = “Ted”
            - Name = “Rebecca”
            - Name = Keeley
    - Constants 
        - Data to never change
            - Let character = “Daphne”
- See Data
    - Print
        - print(character)
- Naming Style
    - Camel case
        - Let managerName
- Tips
    - Prefer constants over names
