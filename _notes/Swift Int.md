title: Swift Int
---

### Main Idea

A basic data type that holds a single whole number.

```swift

let score = 10

// ignores underscores, _, 
let reallyBig = 100_000_000

// Special Operations
counter = 5
counter += 5
counter *= 2
counter -= 10
counter /= 2

// isMultiple(of:)
120.isMultiple(of:3))
```
